Make More Leads
Install the LeadHunter Tampermonkey script to scrape Google Maps directly to your pipeline
⚠️
No Pipeline Script Connected
Contact your admin to set up your LeadHunter pipeline link.
Your account needs a Google Sheets URL in the Users settings.
Step 1
Install Tampermonkey
A browser extension that lets you run custom scripts on any website.
Get Tampermonkey ↗
Step 2
Copy Your Script
Your personal script is pre-configured with your pipeline link. No setup needed.
Step 3
Add to Tampermonkey
Open Tampermonkey → Dashboard → + New Script → paste → Save (Ctrl+S).
Step 4
Scan Google Maps
Go to google.com/maps, search a business type + city, then click 🎯 Scan → Sheets.
Your LeadHunter Script Pre-configured · ready to paste

            Pro tip: Search something like "plumbers Fort Lauderdale" or "HVAC Miami" on Google Maps, then hit Scan. The script auto-scrolls, scrapes all results, filters out chains, Google-verifies each lead has no website, and saves everything directly to your pipeline sheet.
